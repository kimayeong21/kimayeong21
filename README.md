## About Me

안녕하세요. 컴퓨터공학을 전공하며 Python, Java, C, Flask, OpenCV, Arduino를 활용한 프로젝트를 진행해 온 김아영입니다.

저는 단순히 기능을 구현하는 것에서 끝내지 않고, 사용자가 실제로 이해하고 사용할 수 있는 시스템을 만드는 데 관심이 있습니다. 일정 관리, 버스 정보, 영화 추천, 성적 관리, 수질 모니터링, 디지털 유산 관리처럼 생활과 가까운 문제를 기술로 풀어내는 프로젝트를 중심으로 경험을 쌓고 있습니다.

- 서원대학교 IT문화예술대학 컴퓨터공학과 졸업예정

- 관심 분야: AI 서비스, 데이터 기반 시스템, IoT, 웹 백엔드, 사용자 중심 서비스

## Project Highlights

### Digital Legacy Management Service

사용자의 중요한 디지털 정보를 정리하고 관리하는 서비스입니다. 계정, 메모리, 파일, 연락처 등 디지털 유산 관리 흐름을 웹 서비스 형태로 구성하는 데 초점을 둔 프로젝트입니다.

### Smart Schedule Bus System

일정 관리와 대중교통 정보를 결합한 Flask 프로젝트입니다. 사용자는 일정을 등록하고, 청주시 버스 노선과 정류장별 도착 정보를 조회할 수 있습니다. 관리자 기능과 활동 로그 관리도 포함되어 있습니다.

### Smart Water Quality Monitoring System

OpenCV로 웹캠 영상의 색상을 분석하고, Arduino RGB LED로 수질 상태를 표시하는 IoT 프로젝트입니다. 영상 처리, 시리얼 통신, 하드웨어 제어 흐름을 함께 실습했습니다.

### Movie Recommendation System

C 언어로 구현한 콘솔 기반 영화 관리 프로그램입니다. 회원가입, 로그인, 영화 검색, 평점 등록, 관리자 영화 추가·수정·삭제 기능을 CSV 파일 기반으로 구현했습니다.

### Student Grade Management System

Java Swing을 활용한 개인 성적 관리 프로그램입니다. 텍스트 파일의 성적 데이터를 불러와 학기별로 표시하고, 추가·수정·삭제·검색 기능을 제공합니다.
