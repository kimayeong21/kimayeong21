## What I Build

```text
Service Thinking
  └─ 사용자의 문제를 먼저 정의하고 기능 흐름으로 구체화합니다.

Data & Backend
  └─ Flask, SQLite, 파일 입출력을 활용해 데이터를 저장하고 관리합니다.

IoT & Interaction
  └─ OpenCV, Arduino, 시리얼 통신을 활용해 화면 밖의 장치와 연결합니다.

Documentation
```
