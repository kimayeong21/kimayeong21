### 💧 Smart Water Quality Monitoring System

> 한국수자원공사 수질 데이터 + OpenCV 영상 분석 기반 IoT 시스템

- 탁도(tbVal), pH(phVal), 잔류염소(clVal) 데이터 활용
- pH 기준으로 수질 상태 분류 및 LED 출력[pH < 6.5 → 🔴 산성 (RED), 6.5 ~ 7.5 → 🟢 정상 (GREEN), pH > 7.5 → 🔵 알칼리 (BLUE)]
