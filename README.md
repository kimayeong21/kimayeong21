## 💧 Smart Water Quality Monitoring System

> OpenCV 기반 물 색상 분석 + Arduino LED 제어 시스템

- HSV 색상 필터링 기반 수질 상태 분석
- OpenCV 영상 처리 + 시리얼 통신
- 실시간 상태 판단 및 시각화

👉 **핵심 기능**
오염(갈색) → RED BLINK
녹조(초록) → BLUE BLINK
정상(파랑) → GREEN
